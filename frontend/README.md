# Secret Santa Frontend

## Tech Stack:
- React 
- Tailwind CSS

## Tools:
- vite, git

### 3 components:
- Home
- Navbar
- Footer

# Install in local:
git clone <repo>
npm i
npm run dev

# How to use :

Click on the 'Browse Year CSV File' & 'Browse Previous Year Secret Santas CSV File' buttons to upload the csv files.
